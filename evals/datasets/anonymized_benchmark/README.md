# Anonymized Benchmark Dataset

This dataset contains manually curated real-world-style anonymized fixtures for v3.4 blocker rework. It does not contain real names, real employers, emails, phone numbers, addresses, recruitment links, credentials, or private user data. It is a committed benchmark path and schema gate, not external provider proof or external human-review proof.

Counts: 50 JD parser, 30 resume parser, 15 RAG retrieval, 15 RAG answer, 15 match, 15 project rewrite, 15 agent workflow cases; total 155.
