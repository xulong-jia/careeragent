# v3.5 Backup Purge Checklist

- [ ] Privacy deletion proof id is linked to backup purge evidence.
- [ ] Affected backup identifiers are redacted.
- [ ] Backup retention window and legal hold status are recorded.
- [ ] Purged records cannot be restored after completion.
- [ ] Restore block rule is tested for deleted subjects.
- [ ] Audit evidence is stored in private outputs.
- [ ] Legal or compliance reviewer confirms the purge boundary.

Completion boundary: deletion API proof alone is not backup purge proof.
